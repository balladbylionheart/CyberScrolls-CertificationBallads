# 🎵 Scroll VII — The Three Pillars
**SC-900 | CIA Triad**

---

## 📖 Concept Overview

The **CIA Triad** is one of the most fundamental security models in cybersecurity.

It is built upon three core pillars:

- **Confidentiality**
- **Integrity**
- **Availability**

Together, these principles guide how organizations protect information, maintain trust in their systems, and ensure users can securely access resources when needed.

The CIA Triad serves as a foundation for security planning, risk management, and many of the technologies introduced throughout the SC-900 certification.

---

## 🔐 Confidentiality

Confidentiality ensures that information is only accessible to authorized individuals.

Common protections include:

- Encryption
- Access controls
- Authentication
- Data classification

Key question:

👉 Who can see it?

---

## ✅ Integrity

Integrity ensures that data remains accurate, complete, and trustworthy.

Common protections include:

- Hashing
- Digital signatures
- Audit logs
- Change tracking

Key question:

👉 Can it be trusted?

---

## 🌐 Availability

Availability ensures that systems and data remain accessible when needed.

Common protections include:

- Backups
- Failover systems
- Redundancy
- Load balancing

Key question:

👉 Can I access it?

## 🎯 Attack Mapping

Common attack outcomes map directly to the CIA Triad:

| Attack Result | Security Principle |
|--------------|-------------------|
| Data stolen | Confidentiality |
| Data modified | Integrity |
| System offline | Availability |
