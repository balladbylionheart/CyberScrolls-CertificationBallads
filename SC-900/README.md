# 🎓 SC-900 — Security, Compliance, and Identity Fundamentals

**Certification Ballads for Microsoft SC-900**

---

## 📖 Overview

This section of **CyberScrolls: Certification Ballads** is dedicated to the **SC-900 certification**, focusing on core concepts in:

- Security  
- Identity and Access  
- Compliance  
- Microsoft security solutions  

Each concept is transformed into a **scroll** — a combination of music, structure, and technical explanation designed to improve retention.

## 🧠 How to Use This Section

Each scroll is designed to be experienced in two ways:

1. **Listen** — absorb the concept through rhythm and repetition  
2. **Read** — reinforce understanding through structured explanation  

Together, they create a stronger memory connection than traditional study alone.

## 🧭 Learning Focus

The SC-900 path builds understanding of:

- The Shared Responsibility Model  
- Identity and access management (Microsoft Entra)

More scrolls will be added as the journey continues.
